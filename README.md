# InnGuard – Hotel Management System

InnGuard is a web-based hotel management system designed by vithotech for internal use within a local network (LAN). It allows hotel staff to manage bookings, track room availability, handle billing, and monitor daily operations efficiently.

## Features

- Room availability tracking
- Guest check-in / check-out
- Billing and invoice generation
- Dashboard for operational insights
- Responsive UI

## Tech Stack

- React + TypeScript
- Vite

## Getting Started

### Installation
npm install

### Run
npm run dev

## Configuration

to create a .env.local file:

API_KEY= GEMINI_API_KEY
